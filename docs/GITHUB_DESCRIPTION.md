## One-line repo description
DocuParse AI Showcase - FastAPI document parser with OCR fallback, structured extraction, and a readable browser UI for PDF analysis demos.

## About section text
DocuParse AI Showcase is a PyCharm-ready FastAPI project for parsing PDF documents, using OCR for scans, extracting structured fields, and presenting the result in a readable browser report. It is built as a polished demo case for document intelligence workflows and can be adapted for contracts, policies, tenders, and legal documents.
